# ESLint config

An ESLint config for TypeScript projects.

## Install it

```sh
npm i -D eslint @dlevs/eslint-config
```

## Use it

In your `.eslintrc` file:

```json
{
  "extends": "@dlevs/eslint-config"
}
```

If using react, use this instead:

```json
{
  "extends": "@dlevs/eslint-config/react"
}
```
